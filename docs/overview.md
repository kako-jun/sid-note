# sid-note とは何か

## 動機

ベースを弾くとき、楽譜やTAB譜だけでは「どの指でどの弦を押さえるか」が直感的に伝わらない。

TAB譜は弦番号とフレット番号を示せるが、音楽理論との接点がない。
五線譜は音の高さと長さを正確に表せるが、指板上の位置は奏者に委ねられる。

sid-note はこの溝を埋めるために作られた。**「ベースの運指を直感的に記譜したい」**という一点から出発している。

## 何ができるのか

1 音符ごとに、以下の情報をまとめて扱う：

- 音高（ピッチ）と音価（全音符・四分音符・八分音符など）
- 弦番号とフレット番号（左手の指板位置）
- ピッキングの向きと弦（右手の動き）
- そのコードにおける音程（ルート・3度・5度・7度）
- 機能和声（Ⅰ・Ⅱ・Ⅲ・Ⅳ・Ⅴ・Ⅵ・Ⅶ）とカデンツ

## TAB譜との違い

| 観点 | TAB譜 | sid-note |
|------|-------|----------|
| 指板位置 | あり | あり |
| 音楽理論との結びつき | なし | あり（コード・スケール・機能和声） |
| 音高の明示 | なし（フレットから推定） | あり |
| ピッキング情報 | なし | あり |
| 複数のポジション候補 | 手動で選択 | 自動計算で提示 |

## アーキテクチャの方針

- UI は Next.js / React で構成する
- 音楽理論の計算ロジックは TypeScript で実装済み。将来的に Rust/WASM へ移行して高速化する
- データはトラック単位の YAML ファイルで管理する（Zod でスキーマ検証）
- Cloudflare Pages または Docker でホスティングできる
